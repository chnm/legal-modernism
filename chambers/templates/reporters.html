Check citations via reporter_standard

This page lists every distinct reporter_standard value from the legalhist.reporters_citation_to_cap whitelist table. This table maps the many variant abbreviations found in MOML treatises (the reporter_found column) to a standardized reporter name (reporter_standard). The number in parentheses shows how many distinct reporter_found abbreviations map to each standard reporter. Click a reporter to see a sample of raw citations from moml_citations.citations_unlinked whose reporter_abbr matches one of the variant abbreviations for that standard reporter.

Show SQL query 
SELECT reporter_standard, count(*) AS n
FROM legalhist.reporters_citation_to_cap
WHERE reporter_standard IS NOT NULL
GROUP BY reporter_standard
ORDER BY reporter_standard

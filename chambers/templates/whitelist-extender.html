Whitelist extender

Top 100 reporter abbreviations from moml_citations.citations_unlinked that do not appear in the whitelist table (legalhist.reporters_citation_to_cap). The count shows how many citations reference each abbreviation. Adding an abbreviation to the whitelist and reprocessing would allow that many citations to be linked.

Show SQL query 
SELECT cu.reporter_abbr, count(*) AS n
FROM moml_citations.citations_unlinked cu
LEFT JOIN legalhist.reporters_citation_to_cap wl
  ON cu.reporter_abbr = wl.reporter_found
WHERE wl.reporter_found IS NULL
GROUP BY cu.reporter_abbr
ORDER BY n DESC
LIMIT 100
